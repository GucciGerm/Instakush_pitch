Data

Base Model

This will contain data regarding our id, when it was created at, and when it will be updated at.

User Model

This will contain data regarding our user's name, address, username, password, and their phone number (for contact and delivery).

Driver Model

This will contain data regarding our driver's name, location, username, password, and their marks (in case they have any penalties for a canceled delivery)

Dispensary Model

This will contain data regarding a dispensary's name, username, password, and address.

Item Model

This will contain data regarding our image link (for the marijuana), the name of the item (marijuana), the dispensary_id, the description, the price, the company selling the strain, the serving_size being sold of the marijuana, the stock (inventory of marijuana available), and the type of strain being sold.

Order Model

This will contain data regarding the mode that the order is in (i.e. order received, kush ready, and driver on they way), the dispensary_id, and the item_id.
